# tag-news

Sistema ASP.NET MVC com Entity Framework (Model First) e Migrations para realização de CRUD de Noticias e Tags.

### ⚙️ Pré-requisitos

- .NET SDK instalado (versão 8.0)
- PostgreSQL 17.5
- String de conexão ajustada no `appsettings.json`

---
